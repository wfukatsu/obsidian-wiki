# Wiki Init

Initialize the LLM Wiki vault structure.

## Workflow

1. Create directories if they don't exist:
   - `vault/sources/`
   - `vault/wiki/`

2. Create `vault/schema.md` with the wiki schema definition (page types, conventions, workflows, quality standards)

3. Create `vault/index.md` with category headers: Summaries, Entities, Concepts, Comparisons, Syntheses

4. Create `vault/log.md` with a header for chronological operation records

5. Report the initialized structure

Do not overwrite existing files.
